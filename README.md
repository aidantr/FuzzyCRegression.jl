[![Coverage](https://codecov.io/gh/aidantr/FuzzyCRegression.jl/branch/main/graph/badge.svg)](https://codecov.io/gh/aidantr/FuzzyCRegression.jl)

This package implements the heterogeneous effects estimator from Lewis, Melcangi, Pilossoph, and Toner-Rodgers in Julia. The [documentation](https://aidantr.github.io/FuzzyCRegression.jl/dev/) provides an introduction to the estmimator, describes the FuzzyCRegression.jl package, and shows several examples.

## Installation 

```julia
julia> Pkg.add("FuzzyCRegression")
```
